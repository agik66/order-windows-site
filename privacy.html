Privacy Policy

Order Windows · Effective: May 25, 2026 · Last updated: May 25, 2026

Summary

Order Windows does not collect, store, or transmit any personal data. The app runs entirely on your Mac. We have no servers, no analytics, no tracking.

Data Collection

We do not collect any of the following:

  • Name, email, phone, or any contact information
  • Device identifiers or advertising IDs
  • Usage analytics or telemetry
  • Crash reports (Apple's standard system crash reporter is not enabled)
  • Location data

Local Data Storage

The app stores your preferences (keyboard shortcuts, padding, radial menu size) in NSUserDefaults on your local machine. These preferences never leave your device unless you have iCloud Keychain Sync enabled — in that case, Apple synchronizes them; we have no access.

Accessibility Permission

Order Windows requires macOS Accessibility permission (System Settings → Privacy & Security → Accessibility) to read and set window frames of other applications. This permission is used only for that purpose. No window content is captured or transmitted. All Accessibility API calls happen locally and synchronously when you trigger a shortcut or radial menu action.

Network Usage

Order Windows does not connect to the internet. The app contains no network code. You can verify this with macOS Little Snitch or any network monitoring tool.

Third-Party Libraries

The app uses KeyboardShortcuts by Sindre Sorhus (MIT License) for global keyboard shortcut registration. This library runs locally and does not connect to any external service.

Children's Privacy

The app is not directed at children under 13. We do not knowingly collect any information from children.

Changes to This Policy

If we materially change how the app handles data, this page will be updated and the "Last updated" date will change. The current Mac App Store description always reflects the live behavior.
