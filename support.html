Quick start

  1. Install from the Mac App Store.
  2. On first launch, macOS asks for Accessibility permission. Click Open System Settings, then toggle Order Windows on.
  3. Open any window (TextEdit, Safari, …).
  4. Press ⌃⌥← — the window snaps to the left half.
  5. Hold ⌥Q — the radial menu appears under your cursor.

Troubleshooting

Shortcuts don't move windows. Open System Settings → Privacy & Security → Accessibility. Make sure Order Windows is in the list and toggled ON. If it's off, toggle it on — the app activates within 1 second without restart.
Some windows don't fit (e.g., Music, Mail). Some apps have a minimum window size larger than a quarter of your screen. Order Windows can't override that. Try on a larger display or use halves instead of corners.
App still asks for Accessibility even though I granted it. After an app update, macOS may treat the new binary as a different app. Remove Order Windows from the Accessibility list (− button), then add it back (+ button) and toggle on.
Top-corner snap on macOS Tahoe goes to native tiling instead. macOS 26 has built-in window tiling for the top-left/top-right corners. To use Order Windows snap there exclusively, disable native tiling in System Settings → Desktop & Dock → Windows.
